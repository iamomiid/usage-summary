# Usage Summary

Track API usage and rate limits for OpenRouter and Codex.

## Features

- **OpenRouter Integration**: View credits, 30-day totals, and model-level activity in one place
- **Codex Integration**: Check 5h and weekly limits for multiple Codex accounts

## Setup

1. Install the extension in Raycast
2. Go to Extension Preferences
3. Add your OpenRouter API key (management API key)

## Commands

- **OpenRouter Usage**: Combined view of credits, 30-day totals, and model breakdown
- **Codex Usage**: Combined view of 5h and weekly limits per account

## Setup

1. Install the extension in Raycast
2. Go to Extension Preferences
3. Add your OpenRouter API key
4. Add one or more Codex auth paths, separated by commas or newlines

## API Documentation

- [OpenRouter Credits API](https://openrouter.ai/docs/api/api-reference/credits/get-credits)
- [OpenRouter Activity API](https://openrouter.ai/docs/api/api-reference/analytics/get-user-activity)
